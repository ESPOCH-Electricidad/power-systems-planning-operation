La evaluación del bloque se divide en tres ejercicios progresivos. Cada ejercicio deriva de un modelo revisado, pero exige una variante o extensión para que el estudiante no copie el modelo base.

| Actividad | Modelo relacionado | Tipo |
|---|---|---|
| [01A — Producción lineal de componentes eléctricos](actividad_01A_produccion_lineal.md) | Producción con recursos limitados | PL |
| [01B — Transporte de energía entre fuentes y cargas](actividad_01B_transporte_energia.md) | Transporte de energía | PL |
| [01C — Localización binaria de equipos de monitoreo](actividad_01C_localizacion_binaria.md) | Localización y cobertura | MILP |

## Entregables comunes

- archivo `.dat` construido desde las tablas;
- archivo `.mod` formulado por el estudiante;
- archivo `.run`;
- archivo `.out`;
- archivo Excel con resultados y gráficos;
- informe PDF breve con análisis técnico.
