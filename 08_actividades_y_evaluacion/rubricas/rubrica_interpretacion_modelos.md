# Rúbrica para interpretación de modelos de optimización

| Criterio | Excelente | Bueno | Básico | Insuficiente |
|---|---|---|---|---|
| Formulación | Define correctamente conjuntos, parámetros, variables, función objetivo y restricciones | Presenta la mayoría de elementos con errores menores | Omite elementos importantes | La formulación no corresponde al problema |
| Lectura de datos | Identifica unidades, límites y supuestos del caso | Reconoce los datos principales | Confunde algunas unidades o parámetros | No interpreta los datos de entrada |
| Ejecución | Ejecuta el caso de forma ordenada y reproduce resultados | Ejecuta con pequeñas inconsistencias | Requiere apoyo significativo | No logra ejecutar o documentar el proceso |
| Interpretación | Relaciona resultados con criterios técnicos y económicos | Explica resultados principales | Describe resultados sin análisis suficiente | No justifica conclusiones |
| Sensibilidad | Compara escenarios y explica cambios | Compara escenarios con explicación parcial | Menciona cambios sin suficiente fundamento | No realiza comparación |
| Comunicación | Presenta conclusiones claras, gráficas y bien organizadas | Presenta conclusiones comprensibles | Presenta información poco ordenada | Entrega incompleta o confusa |
