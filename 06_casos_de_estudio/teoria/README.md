Un caso de estudio no es solo un archivo de datos. Es una representación estructurada del sistema que debe indicar qué modelos puede alimentar, qué datos están disponibles y qué supuestos se introducen.

| Categoría | Datos requeridos |
|---|---|
| Buses | demanda, tipo de barra, tensión si aplica |
| Líneas | origen, destino, reactancia, resistencia, límite |
| Generadores | ubicación, costos, límites, tecnología |
| Escenarios | demanda, disponibilidad, costos, inversión |
| Metadatos | fuente, adaptación, unidades, estado |
